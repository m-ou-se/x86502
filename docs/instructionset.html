6502 Instruction Set

All instructions are 1 byte (8 bits). In this document, we call the least significant two bits 'IS' (instruction set), the next three bits 'OM' (operand mode), and the three remaining bits 'II' (instruction index). See the overview table for IS 01 for the clearest example to see why these names are useful.
